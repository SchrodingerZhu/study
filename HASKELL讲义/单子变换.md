# 单子变换

## Kleisli
单子自身也可以构成一个范畴，叫做`Kleisli`范畴。这个范畴的特点是：
- return 是单位态射，对应`Hask`范畴的`id`
- `>=>` 是组合运算，对应`Hask`范畴的`.`

其中`>=>`的定义是
```haskell 
(>=>) :: Monad m => (a -> m b) -> (b -> m c) -> (a -> m c)
f >=> b = \x -> f x >>= g

(<=<) :: Monad m => (b -> m c) -> (a -> m b) -> (a -> m c)
f <=< g = flip (>=>)
```
